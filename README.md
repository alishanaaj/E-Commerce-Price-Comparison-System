🛒 E-Commerce Price Comparison System (Price Wise)
📌 Overview

This project is a fully developed E-Commerce Price Comparison System that helps users compare product prices and find the best deals efficiently. It uses core Design and Analysis of Algorithms (DAA) concepts to ensure fast processing and optimal results.

🚀 Features
🔍 Product search functionality
📊 Price comparison across products
⚡ Fast searching using Binary Search
📈 Efficient sorting using Merge Sort & Quick Sort
💡 Greedy approach for best price selection
🎨 User-friendly and responsive interface
🧠 Algorithms Used
Merge Sort
Quick Sort
Binary Search
Greedy Algorithm
🛠️ Technologies Used
Programming Language: C
Frontend: HTML, CSS
Data Structures: Arrays, Lists
⚙️ How It Works
User enters product name
System retrieves product data
Data is stored in arrays/lists
Sorting algorithms arrange prices
Binary Search helps in fast lookup
Greedy approach selects the best price
Results are displayed to the user
